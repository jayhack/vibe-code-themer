On the quietness of well-set type.

A magazine, like a building, is built from a grid. Twelve columns, a baseline, and a few firm rules — and the page suddenly knows what to do with itself.

— The Editors

Feature

1 The grid as governance

pp. 12 — 27

By

M. Brockmann

Zürich, 2026

G ood typography is a consequence of structure, not an addition to it. The page is laid out before a single word is placed; the words then find their seats. There is nothing mysterious about this. It is, simply, a contract between writer and reader.

The grid is not decoration. The grid is the governance under which decoration becomes unnecessary. A column begins where the previous column ends. A baseline holds firm. A headline announces itself with size and weight, never with ornament. Restraint is the loudest tool we have.

Coding agents, given the same restraint, produce interfaces that look like furniture instead of fireworks: things you can live with, things that hold weight. Set a 12-column grid. Pick one sans, one serif. Allow one accent — red, by tradition. The rest is editing.
