III
CHAPTER · HALO CALIBRATION

CALIBRATING
THE four HALOES

The four halo lamps are not decoration. Each is a 5000K ring of 38 individually-bonded LEDs sitting behind a polished chrome bezel. The driver-side inner lamp carries the high-beam projector; the outer three carry low-beam, cornering, and signature. Do not align by eye. Use the jig.

"The car arrives lit from inside. Set the lamps to nod, not to stare." — FOREMAN'S RULE · G. ROSSI · 1998
III.2 · ALIGNING THE BEZEL

The bezel ring sits 1.4mm proud of the body panel. Anything more reads as fitment error from 3m; anything less collects road grit at the joint. Use the calliper and the torque sheet — never thumb-feel.

SPECIMEN / 14 
// manual/calibrate-halo.ts
import { HaloRing } from "@prowler/lamp"

export const calibrate = (bay: number) => {
  const jig = HaloRing.align({
    ring:    "chrome-bezel-38mm",
    leds:    38,         // per lamp
    kelvin:  5000,
    proud:   1.4,        // mm above panel
    torque:  8.4,        // Nm · ± 0.2
  })
  return jig.seat(bay)
}
