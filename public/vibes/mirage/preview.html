III
CHAPTER · LENS CALIBRATION

CALIBRATING
THE crimson LENS

The crimson lens is not a UI element. It is a discipline. Calibrate it once, at the start of every expedition, and every subsequent decision becomes a matter of "is this a reticle, or a flood?" If it floods, you have not earned the red.

"The lens does not see better than the eye. It sees less, but at a temperature the dune respects." — FIELD NOTE · ABRA HALLIN · 1978
III.2 · TUNING THE RETICLE

The reticle is a 14–22px composite glyph: emerald ring, red crosshair, red center diamond. Anywhere a list, an alert, or a section header would normally take an arrow or a check, take a reticle instead.

◆ SPECIMEN / 14 
// codex/calibrate.ts
import { Reticle } from "@mirage/lens"

export const calibrate = (input: string) => {
  const lens = Reticle.tune({
    ring:    "#1FA956",  // emerald
    cross:   "#E0303F",  // crimson
    center:  "#E0303F",
    bearing: 217,
  })
  return lens.scan(input)
}
